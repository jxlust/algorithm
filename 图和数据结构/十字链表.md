### 十字链表

使用十字链表压缩存储稀疏矩阵时，矩阵中的各行各列都各用一各链表存储，与此同时，所有行链表的表头存储到一个数组（rhead），所有列链表的表头存储到另一个数组（chead）中。

两个指针域分别用于链接所在行的下一个元素以及所在列的下一个元素。

![演示](./olnode.jpg)

类如下

```ts
class OLNode{
	i: number;//行列值
	j: number;
	value: number;//元素的值
	right: OLNode;//两个指针/引用
	down: OLNode;
	constructor(){
	}
 
}
```
